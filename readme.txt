## Projet Rattrapage Architecture Orienté Objet sur le thème de la chaîne de restauration de Mongoo

Pour installer le projet :
1- Cloner le projet
2-Faites la commande dans le terminal ``php --ini --ri pdo_mysql``
3-Après, installer le composer (donc la commande dans le terminal) ``composer require vlucas/phpdotenv``
4- Pour le .env, j'ai mis ma base de donnée laragon. Donc dans le point .env faire :
``db_host=""
db_user=""
db_pwd=""
db_name=""``
5- N'oubliez pas de mettre .gitignore et de mettre le .env et le vendor
6- Tout est bon !
